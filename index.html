🔥 Configurar RifaFácil

Você precisa de um projeto Firebase gratuito para armazenar os dados online e compartilhar com todos. Siga os passos abaixo — leva menos de 5 minutos!

1 Criar projeto no Firebase

Acesse console.firebase.google.com, clique em "Adicionar projeto", dê um nome (ex: minha-rifa) e clique em Criar.

2 Ativar Firestore Database

No menu lateral, clique em Build → Firestore Database → Criar banco de dados. Escolha "Iniciar no modo de teste" (funciona por 30 dias, depois configure as regras).

3 Ativar Storage

No menu lateral, clique em Build → Storage → Começar → Modo de teste → Concluir. Isso permite o upload de fotos do prêmio.

4 Pegar as credenciais do app

Vá em ⚙️ Configurações do projeto → Geral → Seus apps → </> (Web). Registre o app com qualquer nome, e copie o objeto firebaseConfig que aparecer. Cole os valores nos campos abaixo:
