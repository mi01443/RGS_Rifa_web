📄 config.json — Conexão Automática

O arquivo config.json fica no repositório do GitHub junto com o index.html. Quando qualquer pessoa abre o app pela primeira vez, ele lê este arquivo e conecta automaticamente — sem precisar digitar nada.

Conteúdo do config.json:
{
  "apiUrl": "..."
}

Como atualizar o config.json:
1. Copie o conteúdo acima
2. No GitHub, abra o arquivo config.json
3. Clique em ✏️ editar → cole o conteúdo → clique em Commit changes

📋 Como hospedar no GitHub Pages

1. Crie um repositório público no GitHub (ex: minha-rifa)
2. Faça upload dos dois arquivos:
   • index.html (renomeie o rifa.html)
   • config.json (com a URL do Apps Script)
3. Vá em Settings → Pages → Branch: main → Save
4. Seu link: https://seu-usuario.github.io/minha-rifa
5. Quando mudar o Apps Script, só atualiza o config.json
